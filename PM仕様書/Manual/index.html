    実行方法(実行ファイルの動かし方)

        実行するためのコマンドは以下の通りである。
    java -jar forest.jar
    MAC OX Xの場合には、アプリケーションForest.appをダブルクリックする方が簡便である。
    
    
    Forest.app
    
    
    
    また、openコマンドでForest.appを開いても実行可能である。
    $ open Forest.app
